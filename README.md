## üî¨ Compilers, Interpreters & VMs

**Compilation Pipeline:**
- **Lexical Analysis** ‚Äî Tokenization, finite automata (Flex)
- **Parsing** ‚Äî LL/LR parsers, AST construction (Bison, ANTLR)
- **Semantic Analysis** ‚Äî Type checking, symbol tables
- **Code Generation** ‚Äî IR design, optimization passes
- **Runtime Systems** ‚Äî Memory management, GC, JIT

**Virtual Machine Architecture:**
- Stack-based and register-based VM design
- Bytecode instruction sets and encoding
- Built **LC3VM** ‚Äî Complete LC3 virtual machine

**Language Design:**
- Formal grammar specification (BNF/EBNF)
- Type system design and implementation
- Built **Logo4J** ‚Äî Logo language interpreter
